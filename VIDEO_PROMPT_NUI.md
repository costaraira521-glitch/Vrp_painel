## English Version

Create a demonstration video of the NUI interface of the FiveM admin panel "VRP Admin Panel v2.6", modern dark UI style with blue/purple gradients, responsive for desktop and mobile.

Sequence:
1. Login screen: user_id field, password, "Login" button, "VRP Admin Panel" logo at the top, dark background with neon effects.

2. Home dashboard: activity graph (blue line), widgets - players online (number), active bans, pending reports, CPU usage, uptime. Tabs: Home, Admin Logs, Whitelist, Groups, Tools, Admin Chat, Announcements.

3. Tools tab active: 8-card grid:
   - Card 1: "Find Player" with ID input, "Find" button, map preview.
   - Card 2: "Ban/Unban" ban list, ID input, reason field, "Ban" button.
   - Card 3: "Reports" ticket queue, "Resolve", "Refuse" buttons.
   - Card 4: "Server Stats" CPU, memory, players, uptime bars.
   - Card 5: "Permissions" role dropdown, permission checkboxes, "Save" button.
   - Card 6: "Economy" player ID input, balance field, "Set" button.
   - Card 7: "Weather/Time" weather dropdown (sunny, rainy), time slider, "Apply" button.
   - Card 8: "Cleanup DB" checkboxes for logs, reports, bans, "Clean" button.

4. Admin Chat tab: chat box with red/green messages, message input, "Send" button.

5. Announcements tab: announcement text field, "Create Announcement" button, pop-up notification "Announcement sent to all players".

Animations: smooth transitions between tabs, click effects, tooltips, "Operation completed successfully" feedback.

Style: dark theme, futuristic typography, neon blue/purple, game overlay aesthetic, mobile responsive.

Duration: 30 seconds, 1080p 30fps, audio: UI clicks + electronic background.
